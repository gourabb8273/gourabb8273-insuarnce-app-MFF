# Container (Host App)

Host application that loads the Insurance Details and Premium Payment micro-frontends via Webpack Module Federation. It shows selected policy state and renders both remotes in card-style sections.

## How to run

```bash
npm install
npm start
```

Runs at **http://localhost:3000**. Start the remotes (Insurance Details on 3001, Premium Payment on 3002) first if you want full functionality.

## Build

```bash
npm run build
```
